================================================================================
功能描述
================================================================================
本样例主要展示异步串口(LPUART)使用RCL 38400时钟轮询方式收发数据

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA01    TX                      串口调试工具RX
PA02    RX                      串口调试工具TX

辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
串口调试工具
波特率9600bps、8位数据、偶校验、1位停止位

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、本样例接收串口调试工具发送的RX_TX_FRAME_LEN字节数据，并将接收的数据回传给串口调试工具
  （程序中由RX_TX_FRAME_LEN定义了发送和接收的数组长度）
