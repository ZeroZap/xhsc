功能描述
================================================================================
本样例主要展示异步串口(LPUART)进入低功耗后中断方式收发数据，波特率使用XTL生成9600bps

================================================================================
测试环境
================================================================================
测试用板
--------------------------------------------------------------------------------
SK_L021_TSSOP20_Rev1.0

管脚    作用                    测试连接
--------------------------------------------------------------------------------
PA12    USER KEY
PA08    TXD                     串口调试工具RXD
PA09    RXD                     串口调试工具TXD

辅助工具
--------------------------------------------------------------------------------
无

辅助软件
--------------------------------------------------------------------------------
串口调试工具
波特率9600bps、8位数据、偶校验、1位停止位

================================================================================
使用步骤
================================================================================
1、准备测试环境，打开样例工程编译、下载后运行
2、代码运行后，按下USER KEY，系统进入深度休眠
3、串口调试工具发送1字节数据0xXX，唤醒系统，MCU返回0xXX
